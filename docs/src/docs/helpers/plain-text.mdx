# getPlainText

You can extract the plain text of your current state for some custom usecases like
generating content summary.

## Usage

Use `getPlainText` utility to extract the plain text of your content .

```ts
getPlainText( state: State ) => string
```

## Example

An example of `getPlainText` usage:

```ts
import { useState, useEffect } from 'react';
import { EditorState } from 'draft-js';
import { Editor, getPlainText } from 'contenido';


const Demo = () => {
  const [editorState, setEditorState] = useState(
    EditorState.createEmpty()
  );

  useEffect(() => {
    console.log(getPlainText(editorState))
  }, [editorState])

    <>
      <Editor

    </>
  )
}
```
