# Installation

You can use all utilities in one go and there is no need to install a package or a plugin for every single need.

## Guide

You can add `contenido` to your project by running the following commands:

TypeScript:

```shell
npm i draft-js @types/draft-js contenido@latest
```

or JavaScript:

```shell
npm i draft-js contenido@latest
```

## Peer dependencies

`react` >= 18.0.0

`react-dom` >= 18.0.0

`draft-js` >= 0.11.0
