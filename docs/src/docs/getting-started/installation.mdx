# Installation

You can use all utilities in one go and there is no need to install a package for every single need.

## Guide

You can add `contenido` to your project by running the following commands:

```shell
npm i contenido@latest
```

or

```shell
yarn add contenido
```

## Peer dependencies

`react` >= 17.0.0 and `react-dom` >= 17.0.0
